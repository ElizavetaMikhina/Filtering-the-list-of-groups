# Проект "Список групп"

Этот проект представляет собой простое веб-приложение для отображения списка групп с возможностью фильтрации. Приложение написано на языке TypeScript с использованием библиотеки React.

## Установка и запуск

1. Убедитесь, что у вас установлен Node.js и npm.
2. Клонируйте репозиторий проекта с помощью команды `git clone <URL_репозитория>`.
3. Перейдите в каталог проекта: `cd <название_папки_проекта>`.
4. Установите зависимости, выполнив команду `npm install`.
5. Запустите приложение с помощью команды `npm start`.

## Структура проекта

- **src**: Этот каталог содержит исходный код приложения.
  - **components**: В этом каталоге находятся компоненты React.
    - **GroupFilters.tsx**: Компонент фильтров для списка групп.
    - **GroupItem.tsx**: Компонент для отображения отдельной группы.
    - **GroupList.tsx**: Компонент для отображения списка групп.
    - **FriendList.tsx**: Компонент для отображения списка друзей.
  - **App.tsx**: Основной компонент приложения.
- **groups.json**: Файл данных, содержащий информацию о группах.
- **webpack.config.js**: Конфигурационный файл webpack для сборки проекта.

## Использование

После запуска приложения вы увидите список групп. Вы можете использовать фильтры для настройки отображаемых данных: выбрать тип приватности групп, цвет аватарки и указать, показывать ли только группы с друзьями.
